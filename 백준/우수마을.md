# 싱황정리
## 1. 방향성이 없는 트리 (dfs/bfs, union-find, lca, depth, parent array, 트리 DP)
## 2. 인접한 마을들을 동시에 우수 마을로 선정할 수 없다: 하나가 선택되면 인접한 것은 선택되지 않는다 (최대 독립 집합) -> 트리 DP
